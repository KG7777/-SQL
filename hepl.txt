Отправить из гит в локальную папку
1.Необходимо сделать соединение облачного репозитория и локальной папки.
 Для этого вернемся на сайт, зайдем в облачный репозиторий и нажмем на зеленую кнопку code. 
Скопируем путь local
2.Далее перейдем в терминал в Linux и перейдем в нашу созданную папку. 
Для этого необходимо написать команду cd и указать тот путь, где находится наша папка. 
3.Далее необходимо использовать команду git clone и указать после путь, 
который мы скопировали в пункте 1. 
 git clone https://github.com/--/test.git. 

Отправить из локального в гит
1. Перейти в терминале в папку проекта - в ней должна быть папка .git
git add .
git commit -m "Initial commit"
git push -u origin main
